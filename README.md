# 🙋 Jogo do Amigo Secreto 🙋‍♀️
Adicione quantos nomes desejar, depois clique em sortear e descubra qual é o seu amigo secreto!

# 💡Funcionalidades
- Adicionar participantes no campo de texto;
- Armazenar temporariamente os nomes;
- Sorteio aleatório;
- Interface simples.
